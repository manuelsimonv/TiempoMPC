en matlab_der_quadprog.rar se encuentra la simulacion de MATLAB/Simulink original, para hacerla funcionar se debe ejecutar InitMain.m y luego ejecutar SIM_DNNS.slx

en matlab_der_osqp.rar se encuentra la simulacion de MATLAB/Simulink con el reemplazo del solver quadprog por OSQP, para hacerla funcionar se debe ejecutar InitMain.m y luego ejecutar SIM_DNNS.slx. Dentro de este archivo comprimido se pueden encontrar 2 archivos extras, estos son osqp_solve_wrapper_MPC1.txt y osqp_solve_wrapper_MPC2.txt. En estos archivos se pueden modificar los parametros del solver OSQP.
